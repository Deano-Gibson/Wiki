# Experience Lessons

This section turns previous experience into reusable judgment.

It is for sanitized lessons and patterns, not diary-style notes or private internal context.

## Lesson Structure

Use this shape for each lesson:

```md
# Lesson Title

## Context Pattern

## Problem Type

## What Worked

## What Failed

## Default Recommendation

## Reusable Takeaway
```

## Rules

- Remove client secrets and private business details.
- Avoid identifiable internal conflicts.
- Keep emotional/raw notes in private spaces outside published docs.
- Focus on reusable patterns, decision defaults, and warning signs.
