# Personal Knowledge Docs

This site is the curated layer of a personal knowledge system for reusable engineering, delivery, documentation, systems, tools, templates, and sanitized lessons learned.

It is not a raw notes dump. Rough thinking starts in `notes/`, useful material is refined, and only durable public-safe knowledge is promoted into `docs/` for publishing.

## Use This Site For

- Starting a new project with a tested process
- Choosing a stack or tool category
- Reusing templates, checklists, and standards
- Pulling lessons from previous experience
- Keeping project delivery consistent

## Main Entry Points

- [Start Here](start-here.md): project-start workflows and common paths
- [Knowledge Map](knowledge-map.md): full index of the knowledge system
- [Templates](templates/index.md): reusable documents and checklists
- [Decisions](decisions/index.md): stack and architecture decision guides
- [Documentation Standards](documentation/documentation-standards.md): writing and publishing rules
- [Engineering Playbook](engineering/engineering-playbook.md): engineering categories and tool layers

## Operating Model

```text
Obsidian capture -> notes/ working area -> docs/ curated content -> MkDocs Material -> hosted static site
```

The published site is intentionally curated so it stays useful under pressure.
