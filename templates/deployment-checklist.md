# Deployment Checklist

- [ ] Production environment configured
- [ ] Secrets added through platform secret manager
- [ ] Domain and DNS checked
- [ ] Build succeeds
- [ ] Tests pass
- [ ] Database migrations reviewed
- [ ] Rollback path known
- [ ] Error tracking active
- [ ] Uptime check active if needed
- [ ] Logs accessible
- [ ] Critical user flows manually verified
- [ ] Handover notes updated
