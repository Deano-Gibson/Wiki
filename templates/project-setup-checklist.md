# Project Setup Checklist

- [ ] README explains purpose, setup, and commands
- [ ] `.env.example` exists without secret values
- [ ] Dependency installation documented
- [ ] Local development command documented
- [ ] Build command documented
- [ ] Test command documented
- [ ] Lint/format command documented
- [ ] Deployment target selected
- [ ] CI configured
- [ ] Error tracking selected if production-facing
- [ ] Ownership and handover expectations clear
- [ ] License or usage constraints documented
