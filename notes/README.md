# Working Notes

This folder is for Obsidian capture, drafts, research, and working thinking.

MkDocs does not publish this folder.

Recommended flow:

```text
inbox -> drafts or working -> docs
```

Rules:

- Keep notes public-safe if they are tracked in Git.
- Put private material outside this repo or in ignored folders.
- Wiki-links are allowed here.
- Normalize important links when promoting material into `docs/`.
